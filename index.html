This API allows you to extract text from uploaded documents such as PDFs, DOCX, and PPTX.

Usage

Send a POST request to /api/extract with a multipart/form-data body containing a file field named file.

Example Request

curl -X POST -F "file=@yourfile.pdf" https://extract-text-from-file.vercel.app/api/extract

Supported MIME Types

  • application/pdf
  • application/vnd.openxmlformats-officedocument.wordprocessingml.document
  • application/vnd.openxmlformats-officedocument.presentationml.presentation
  • application/vnd.openxmlformats-officedocument.spreadsheetml.sheet
  • text/plain
